# 📖 **Bookshelf Mastery**  
**A Sleek & Modern CRUD App for Book Lovers**  

**Turn your book collection into a beautiful digital library** — built with **PHP 8 + PDO** for rock-solid security and lightning-fast performance.  

✨ **[🚀 Try the Live Demo Now!](https://books-info.kesug.com/)** ✨  

---

### 🌟 **Why You'll Love It**
A clean, dark-themed masterpiece that feels like a premium reading app.  
Manage books with style — upload stunning covers, search instantly, and enjoy buttery-smooth interactions.  

Perfect for bookworms, developers, or anyone who wants to see what a **simple yet stunning** CRUD project can look like.

---

### ✨ **Features That Shine**
- **Full CRUD Magic** — Create, Read, Update, Delete books with **PDO Prepared Statements** (SQL injection? Never heard of it)
- **📸 Book Cover Uploads** — Drag-and-drop style image handling with live previews
- **Smart Validation** — Real-time checks using `empty()` + elegant error messages
- **📱 Fully Responsive** — Looks gorgeous on phones, tablets, and desktops
- **🌙 Modern Dark UI** — Built with **Bootstrap 5** for that premium Netflix-vibe feel
- **Lightning Fast** — Clean code, optimized queries, and beautiful animations

### 🗄️ **Database Structure**
| Field           | Type         | Description                      |
|-----------------|--------------|----------------------------------|
| `id`            | INT (AI)     | Unique book ID                   |
| `title`         | VARCHAR(255) | Book title                       |
| `author`        | VARCHAR(150) | Author name                      |
| `genre`         | VARCHAR(100) | Book genre                       |
| `year_published`| YEAR         | Publication year                 |
| `pages`         | INT          | Total pages                      |
| `image`         | VARCHAR(255) | Cover image filename             |
| `created_at`    | TIMESTAMP    | Auto-generated timestamp         |

**Table Name**: `books`
